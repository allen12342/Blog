Objectif

Vous êtes un méchant et votre rêve est de vous débarrasser de ces super-héros masqués ennuyeux, portant des pantalons de yoga et bizarres. Vous n’avez jamais compris pourquoi certains d’entre eux sont considérés comme des super-héros, simplement parce qu’ils sont riches. D’autres vous ennuient avec leurs discours philosophiques. Nous avons trouvé des informations confidentielles sur ces super-héros. Votre tâche consiste à créer une page Web pour organiser toutes les données sur ces smartypants. Toutes ces données peuvent être trouvées ici dans all.json. Vous devez écrire tout le code à partir de zéro. Vous n’êtes pas autorisé à vous fier à des frameworks ou à des bibliothèques comme React, Vue, Svelte, etc. Récupération des données Pour obtenir les informations, vous devez utiliser fetch. Lorsque vous utilisez fetch dans JS, il renvoie toujours une Promise. Nous approfondirons cela plus tard.
